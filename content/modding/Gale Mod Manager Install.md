Gale Mod Manager is another third party Thunderstore mod manager designed to be an improvement on R2ModMan.  Download the for your system from the [Gale Website](https://kesomannen.com/gale). Once downloaded run the .msi file. You will receive a similar warning to what was given when first installing r2modman.

![[smartscreen_warning.png]]

Follow the installer to completed the installation and launch. On open you will be asked to select a game.

![[gale_game_selection.png]]

If coming from R2ModMan it should recognize any current packs you currently have installed. You can choose to import or skip as you wish.

![[gale_r2modman.png]]

On the next screen you will be asked where you would like your settings to be located. The default location is fine.

To import a pack you can use the Import menu above and select from Code.

![[gale_import.png]]

Select to import as either a new profile or update an existing. Once completed you can launch the game from the top left launch button.

![[gale_import_code.png]]
