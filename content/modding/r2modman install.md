Download the following from [Thunderstore](https://thunderstore.io/package/ebkr/r2modman/) and install it on your system. Use the manual  download option. You may ignore the warning when installing as they just didn't pay for a certificate from Microsoft.

Click more info and then select **Run Anyway**

Once open, search for the game you would like to manage. Click **Select Game**.

![[r2modman-select-game.png]]

Click **Import/Update**

![[r2modman-import.png]]

Click **From Code** and copy and paste the code below.

![[r2modman-from-code.png]]

![[r2modman-code.png]]

On the next window select **Import**

![[r2modman-import-confirm.png]]

If you are updating a profile choose that option if not select **Import new profile**

![[r2modman-profile.png]]

Click **Start Modded** to launch your modded instance of the game and then click **Continue**.

![[r2modman-start.png]]

Codes will look like the REPO one below. For additional codes check out our [[mod collections]] page with the full list.

**R.E.P.O:** `01965b60-842d-c8db-725c-46d165a58d9e` (This will most likely not be current use the link above.)
