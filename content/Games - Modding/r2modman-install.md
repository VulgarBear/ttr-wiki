Download the following from [Thunderstore](https://thunderstore.io/package/ebkr/r2modman/) and install it on your system. Use the manual  download option. You may ignore the warning when installing as they just didn't pay for a certificate from Microsoft.

Click more info and then select **Run Anyway**

[![](https://wiki.talkingroom.xyz/uploads/images/gallery/2025-04/scaled-1680-/image-1745290228116.png)](https://wiki.talkingroom.xyz/uploads/images/gallery/2025-04/image-1745290228116.png)

Once open, search for the game you would like to manage. Click **Select Game**.

[![](https://wiki.talkingroom.xyz/uploads/images/gallery/2025-04/scaled-1680-/image-1745288999509.png)](https://wiki.talkingroom.xyz/uploads/images/gallery/2025-04/image-1745288999509.png)

Click **Import/Update**

[![](https://wiki.talkingroom.xyz/uploads/images/gallery/2025-04/scaled-1680-/image-1745288917372.png)](https://wiki.talkingroom.xyz/uploads/images/gallery/2025-04/image-1745288917372.png)

Click **From Code** and copy and paste the code below.

[![](https://wiki.talkingroom.xyz/uploads/images/gallery/2025-04/scaled-1680-/image-1745288950763.png)](https://wiki.talkingroom.xyz/uploads/images/gallery/2025-04/image-1745288950763.png)

On the next window select **Import**

[![](https://wiki.talkingroom.xyz/uploads/images/gallery/2025-04/scaled-1680-/image-1745290321741.png)](https://wiki.talkingroom.xyz/uploads/images/gallery/2025-04/image-1745290321741.png)

If you are updating a profile choose that option if not select **Import new profile**

[![](https://wiki.talkingroom.xyz/uploads/images/gallery/2025-04/scaled-1680-/image-1745290369774.png)](https://wiki.talkingroom.xyz/uploads/images/gallery/2025-04/image-1745290369774.png)

Click **Start Modded** to launch your modded instance of the game and then click **Continue**.

[![](https://wiki.talkingroom.xyz/uploads/images/gallery/2025-04/scaled-1680-/image-1745289576517.png)](https://wiki.talkingroom.xyz/uploads/images/gallery/2025-04/image-1745289576517.png)

Codes will look like the REPO one below. For additional codes check out our [page](/game-modding/ttr-mods) with the full list.

**R.E.P.O:** `01965b60-842d-c8db-725c-46d165a58d9e` (This will most likely not be current use the link above.)
